Explain to your non-tech friend

.map()The map method takes a group of element of an Array and performs a set of conditions (function) to each item in the array in the order that they appear, then takes the results that that process generates to form a new array.

.filter() Similar to the map method, the filter method takes a group of items (an array) and performs a set of conditions (function) to each item in the array in the order that they appear, but then takes only the results that pass/agree with the set of conditions that process generates to form a new array.
