What 'scope' is, and how it works in JavaScript.

In Javascript, scope refer you to the value or data of your context of your code, and there are two scpoes, Local and Global, Local scope is a variable inside the function, any code outside a function can't access this variable/ and Global is A variable outside the function, in global scope any code inside a function can access this variable
